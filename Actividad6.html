Silgas

OSI ( Open Systems Interconnection )

es un modelo conceptual que divide las funciones de comunicación de red en siete capas. Este modelo fue creado por la Organización Internacional para la Estandarización.

POP (Post Office Protocol)

puede referirse a Protocolo de Oficina Postal, Punto de Presencia o Programación Orientada a Procedimientos.

SMYP (Simple Mail Transfer Protocol)

Protocolo para la transferencia de corre electronico en formato simple

IP (Internet Protocol)

Representa la dirección lógica de un dispositivo; esta formada por 4 numeros en un rango de 0 a 255

HTTP ( Hyper Text Transfer Protocol )

Protocolo de transporte de archivos en formato html (Painas Web)

TCP ( Transmission Control Protocol )

puede referirse a Protocolo de Control de Transmisión, túbulo contorneado proximal o tripulante de cabina.

UDP (User Datagram Protocol)

puede referirse a la Unidad de Desarrollo Productivo, el User Datagram Protocol o la uridina difosfato.

SSH (Secure Shell) es un protocolo y un programa que permite acceder a un servidor remoto de manera segura.

FTP (File Transfer Protocol) es un protocolo que se utiliza para transferir todo tipo de archivos entre equipos conectados a una red

SMTP ( Simple Mail Transfer Protocol) Significa protocolo simple de transferencia de correo. Es un protocolo de comunicación que permite enviar y recibir correos electrónicos por internet.
